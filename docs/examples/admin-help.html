Amaze UI 1.0.0 Beta

对于 Amaze UI，我们不是创造者。

Amaze UI 的诞生，依托于 GitHub 及其他技术社区上一些优秀的资源；Amaze UI 的成长，则离不开用户的支持。

感谢开源！感谢有你！

Bug 反馈

感谢对 Amaze UI 的关注和支持，如遇到 Bug 或者使用问题，可以通过以下途径反馈给我们：

  1. 在 GitHub 项目主页提交 Issue。
  2. 在 GitHub 项目主页提交 Issue。
